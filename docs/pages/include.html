Include Statements

Include Statements:

What is an include statement Include statements are used to import both data from both the standard library and from other files How to include something Include statements follow the syntax of #include `original name` from `location` as `name` with `parameters` returns `return value` It is important to note that when including from the standard library you use `<location>` whereas when including from a relative file you use "location". For example, to include the standard library function print from stdio, one would use #include print from <stdio> as print with [str] Example of a full code block using that include 


#include print from <stdio> as print with [str]

export impure fn main = () => {
    str x = "Hello, world!";
    print(x); // Prints "Hello, world!"
}

As you can see, print(str) is included into the file, and it is then used to print a string. Trusting that included values are pure By default, values are considered impure when included from the standard library or included from .tmb files where they are declared as impure. In order to treat a value as pure, use #trust include instead of just #include .
