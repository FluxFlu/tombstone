Numerical Data Types

Numerical data types:

List of numerical data types: - i32: 32 bit signed integer - i64: 64 bit signed integer - f32: 32 bit signed floating point number - f64: 64 bit signed floating point number What does that mean? - A signed number can be positive or negative - An `n bit` value uses n bits to store its data. Effectively, 64 bit numbers have much larger maximum and minimum values. - A floating point number can be a decimal, while an integer cannot.
