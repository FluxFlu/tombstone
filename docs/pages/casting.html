Type Casting:

What is Type Casting Type Casting is used to convert a value of one type to a different type. How to type cast Treat the type as though it were a function. For example: 

// Note the i32 used in the include statement 
#include printNum from <stdio> as printNum with [i32]

f64 x = 12.44;

// Prints 12:
printNum(i32(x));
